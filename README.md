# 22_club_redesign

** Project Overview

This project focuses on the redesign of a local snooker and social club website. The aim of the redesign was to improve the overall user experience by making the site more user-centred, accessible, and easier to navigate. The original website contained several usability issues including poor mobile responsiveness, unclear navigation, inconsistent typography, and weak visual hierarchy. The redesign process followed a structured UX design workflow, beginning with research and analysis of the existing website before moving into design exploration, prototyping, development, and testing.

** Design Goals

The redesign aimed to achieve the following improvements:
- Improve navigation clarity and overall user flow
- Increase readability through better typography and layout
- Improve mobile responsiveness and accessibility
- Create a clearer content hierarchy
- Ensure key information such as facilities, membership, and location is easy to find

** UX Design Process

* Desk Research

The existing 22 Club website was analysed to identify usability issues and areas for improvement. Additional inspiration websites were also reviewed to explore modern design patterns used in similar venues such as snooker clubs, bars, and leisure venues.

* User Personas

Personas were created to represent potential users of the club website. These helped identify user goals, motivations, and pain points when trying to find information such as opening hours, facilities, and membership details.

* Sketches (Crazy 8)

Early layout ideas were generated using sketching techniques to explore different possible structures for the homepage and main content sections.

* Low Fidelity Wireframes

Low fidelity wireframes were created to define the structure of the pages and organise content before focusing on visual design.

* High Fidelity Prototype

A refined prototype was produced to demonstrate how the redesigned interface would appear, including improved typography, layout consistency, and clearer navigation.

* Development

The redesigned website was built using modern web technologies consiting of:
- Semantic HTML5
- CSS3
- JavaScript

The site was developed with responsive design principles to ensure usability across desktop and mobile devices.

* Testing

The website was tested using Google Lighthouse to evaluate:
- Performance
- Accessibility
- Best practices
- SEO

Testing was performed on both desktop and mobile devices. Issues identified during testing were addressed through improvements to layout, semantic structure, and performance optimisation, specifially image compression/file conversion. The website was also tested using Contrast Checker to evaluate the background and text colour to ensure everything met AA and AAA criteria.

* Outcome

The final redesign provides a cleaner and more structured interface that improves readability, navigation, and accessibility. The updated layout presents information more clearly and ensures visitors can quickly find key information about the club.
